# 1.Genspark关联github

Genspark关联Github，生成代码Upload以后，在genspark_ai_developer

把远程genspark_ai_developer的代码拉取到本地main分支的方法
```

# 1. 确保在 main 分支
git checkout main

# 2. 拉取远程最新信息
git fetch origin

# 3. 合并远程分支到本地 main
git merge origin/genspark_ai_develop

```
