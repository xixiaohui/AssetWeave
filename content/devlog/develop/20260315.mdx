下面给你一套 可以直接运行的 Python Skill，它会：

1️⃣ 调用 AI 生成图片
2️⃣ 生成 4 张漫画
3️⃣ 自动拼接四格
4️⃣ 输出 comic_xxxx.png

mkdir -p ~/.openclaw/skills/comic-image

nano comic_image.py

```
import os
import requests
from PIL import Image
from io import BytesIO

# =========================
# 配置
# =========================

API_URL = "https://openrouter.ai/v1/images/generate"
API_KEY = os.getenv("OPENROUTER_API_KEY")

OUTPUT_DIR = "./output"
os.makedirs(OUTPUT_DIR, exist_ok=True)


# =========================
# 生成图片
# =========================
def generate_image(prompt, index):

    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json",
    }

    payload = {
        "model": "",
        "prompt": prompt,
        "width": 1024,
        "height":1024

    }

    r = requests.post(API_URL, headers=headers, json=payload)

    print("API RESPONSE:", r.text)

    data = r.json()

    image_url = data["data"][0]["url"]

    img = Image.open(requests.get(image_url, stream=True).raw)

    return img


# =========================
# 拼接四格漫画
# =========================

def combine_comic(images):

    imgs = [Image.open(i) for i in images]

    w, h = imgs[0].size

    canvas = Image.new("RGB", (w*2, h*2), "white")

    canvas.paste(imgs[0], (0,0))
    canvas.paste(imgs[1], (w,0))
    canvas.paste(imgs[2], (0,h))
    canvas.paste(imgs[3], (w,h))

    path = f"{OUTPUT_DIR}/comic.png"
    canvas.save(path)

    return path


# =========================
# 主流程
# =========================

def generate_comic(prompts):

    images = []

    for i, p in enumerate(prompts):
        img = generate_image(p, i)
        images.append(img)

    comic = combine_comic(images)

    return comic


# =========================
# 测试
# =========================

if __name__ == "__main__":

    prompts = [
        "cartoon father sleeping in bed, daughter beside bed, morning light",
        "little girl with ponytail ringing alarm clock near father",
        "father suddenly waking up shocked, daughter laughing",
        "father and daughter laughing together in living room"
    ]

    comic = generate_comic(prompts)

    print("Comic generated:", comic)
```

# 6.创建 Skill 配置文件

在 comic-image 目录创建：

skill.yaml

路径：

~/.openclaw/skills/comic-image/skill.yaml

内容：
```
name: comic-image
description: Generate comic images using OpenRouter and combine into a comic strip
version: 1.0

entrypoint: comic_image.py

inputs:
  - name: prompts
    type: list
    description: list of image prompts

outputs:
  - name: comic_path
    type: string
    description: generated comic image path
```

## 修改 comic_image.py 让 Skill 可调用

OpenClaw Skill 通常从 stdin 接收参数。

在 comic_image.py 最后加：

```python
import sys
import json

def main():

    data = json.load(sys.stdin)

    prompts = data.get("prompts", [])

    comic = generate_comic(prompts)

    result = {
        "comic_path": comic
    }

    print(json.dumps(result))


if __name__ == "__main__":
    main()
```

这样 OpenClaw 才能调用。
