### 🔑 OpenClaw 调用上传 API
```

POST /api/comics/upload
Authorization: Bearer 89d42f249496a09eb9478ed94edbcbf9603e30e02aabe9c6c71af8c14178ca7f
Content-Type: multipart/form-data

Fields:
  comic_title    必填  漫画名称（如 "Dragon's Path"）
  chapter_num    必填  章节号（如 1 或 1.5）
  chapter_title  选填  章节标题
  description    选填  描述
  author         选填  作者（默认 OpenClaw）
  genre          选填  类型（action/fantasy/romance...）
  tags           选填  逗号分隔标签
  comic_slug     选填  自定义slug（不填自动从标题生成）
  pages          必填  图片文件（支持jpg/png/webp/gif，自动转webp）

Response:
  {
    "success": true,
    "comic": { "id", "title", "slug", "url": "/comics/xxx" },
    "chapter": { "id", "chapter_num", "page_count" },
    "pages": [{ "page_num", "image_url" }]
  }

```
```
📊 数据库设计（PostgreSQL）

表	说明
api_keys	API密钥（SHA-256哈希存储）
comics	漫画系列（title/slug/cover/genre/status...）
chapters	章节（关联comic，自动统计page_count）
pages	每页图片（URL + 存储路径 + 尺寸信息）
```

```
图片存储路径规则

                                         0002.webp
                                 ch0002/0001.webp
```
