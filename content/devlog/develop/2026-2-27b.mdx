## 1.一句话总结
```js

| 类型      | 本质      | 更像什么     |
| ------- | ------- | -------- |
| 光伏电站收益权 | 长期现金流资产 | 类 REIT   |
| 商业地产租金  | 长期现金流资产 | 类 REIT   |
| 应收账款    | 短期确定回款  | 类债券      |
| 供应链金融资产 | 短期循环融资  | 类票据 / 保理 |
```


## 2.平台而言
这四类资产 不能混在一起卖。

必须分为两大产品线：

- 产品线 A：短期债权型

应收账款

供应链金融

特点：

✔ 固定期限
✔ 本金清晰
✔ 固定收益
✔ 周转快

风险在于信用审查。

- 产品线 B：长期分红型

光伏

商业地产

特点：

✔ 长期现金流
✔ 永续或长周期
✔ 市场退出
✔ 无到期刚兑

风险在资产价格。


```js
建议分两个板块：

# 固定收益专区（短期）

3个月 / 6个月 / 1年
类债权

# 资产分红专区（长期）

永续
类REIT

不要混合。
```

## 3.光伏电站收益权

融资逻辑

前期建站投入大

之后长期卖电

电费来源稳定（如果是政府补贴更稳）

现金流：

每月卖电收入

持续 15~25 年

本质

👉 长期基础设施资产

适合：

永续分红型

5~10年结构

不适合：

1年短期保本产品

风险

政策补贴变化

发电量波动

设备老化

## 4.商业地产租金


融资逻辑

房子已经建好

出租产生现金流

长期持有

现金流周期：

8~15 年回本

本质

👉 长期现金流资产

和光伏非常类似。

## 5.应收账款

融资逻辑

例子：

企业卖货给大公司

账期 90 天

现在缺钱

把这笔应收账款转让给投资人。

投资人 90 天后收回本金 + 利息。

本质

👉 短期确定回款资产
👉 类债券

现金流：

一次性回款

适合：

3个月 / 6个月产品

固定收益

本金回收明确

## 6.供应链金融资产

融资逻辑

核心企业信用背书

上游供应商提前拿钱

下游到期付款

本质类似：

👉 保理 + 票据

通常：

30~180 天周期

高周转

风险取决于：

核心企业信用

贸易真实性

## 7.核心差异

### 期限差异
```js
| 资产    | 期限      |
| ----- | ------- |
| 光伏    | 10~25年  |
| 地产    | 8~20年   |
| 应收账款  | 30~180天 |
| 供应链金融 | 1~6个月   |


```

### 本金来源差异
```js
| 资产    | 本金从哪里来    |
| ----- | --------- |
| 光伏    | 资产出售或长期持有 |
| 地产    | 资产出售或再融资  |
| 应收账款  | 到期回款      |
| 供应链金融 | 到期付款      |

```
👉 应收账款本金来源最清晰
👉 地产和光伏本金依赖资产处置
### 风险结构差异

```js
| 类型    | 风险点      |
| ----- | -------- |
| 光伏    | 发电量 + 政策 |
| 地产    | 空置率 + 房价 |
| 应收账款  | 买方违约     |
| 供应链金融 | 贸易造假     |

```
